"Таксі" - Landing page для надання послуг таксі з різноманітними функціями для зручності клієнтів.

Цей проект включає створення сучасної та зручної веб-сторінки для сервісу таксі, де клієнти можуть легко обрати автомобіль відповідно до кількості пасажирів та вантажопідйомності. Всі ці параметри інтегровані у форму повідомлення для перевізника, що дозволяє оперативно отримувати та обробляти заявки.
- Зручний інтерфейс для вибору автомобіля
- Налаштування під кількість пасажирів та вантаж
- Вибір форми оплати та валюти
- Інтегрована система повідомлень для перевізників
